## 比赛背景

根据提供的数据，对木薯叶图像进行 <b>Cassava Bacterial Blight (CBB)</b>，<b>Cassava Brown Streak Disease (CBSD)</b>，
<b>Cassava Green Mottle (CGM)</b>，<b>Cassava Mosaic Disease (CMD)</b>和<b>Healthy</b>共5个类别的图像分类。
